Fraud Detection

In this project, the focus is on Fraudulent transactions posing an increasing challenge for fintech companies. A dataset of over 51,000 transactions is what i used which is labeled as either fraudulent or legitimate, reflecting real-world patterns.

Project Overview

This project addresses the growing challenge of fraudulent activities faced by fintech companies. Using a dataset of over 51,000 transactions labeled as either legitimate or fraudulent, I applied data analysis and machine learning techniques to uncover suspicious patterns and build an effective fraud detection model

Technologies Used

  • Pandas, NumPy – Data manipulation
  • Scikit-Learn-Machine learning and model evaluation
  • Matplotlib/Seaborn- Data visualization
  • Jupyter Notebook – Interactive analysis

Key Features

  • Transaction amount, time, and location
  • User ID and behavioral patterns
  • Payment methods and account types
  • Device Types

Achievement

This project improved my ability to handle imbalanced datasets, design robust classification systems, and apply data science techniques to a critical real-world problem. It also strengthened my skills in Python and libraries like Scikit-learn, Pandas, Matplotlib, and Seaborn.
