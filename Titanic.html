Survival At Sea

This project analyzes the Titanic dataset to explore and predict the survival of passengers based on various features such as age, gender, class, and more.

Project Overview

This project involved performing exploratory data analysis (EDA) on the famous Titanic dataset to understand the factors influencing passenger survival. I examined various demographic and travel-related variables to uncover patterns and trends. Following the EDA, I built and evaluated predictive models to estimate survival probabilities.

Technologies Used

  • Pandas, NumPy – Data manipulation
  • Scikit-Learn-Machine learning and model evaluation
  • Matplotlib/Seaborn- Data visualization
  • Jupyter Notebook – Interactive analysis

Key Findings

  • Female passengers had significantly higher survival rates than males.
  • Passengers in first class were more likely to survive than those in lower classes.
  • Age and fare were moderately correlated with survival.

Outcome

This project sharpened my data exploration skills, strengthened my ability to draw actionable insights from complex datasets, and improved my proficiency with Python-based data analysis libraries. .
